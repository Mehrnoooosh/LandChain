
Installing and running the Landchain 
Prerequisites:
•	Node.js
•	NPM
•	Express.js
•	Mongo database
•	React.js
•	Solidity
•	Web3.js
Clone the repository to your local machine, move to the ClientSide folder, run npm install
git clone https://github.com/Mehrnoooosh/LandChain.git

cd ClientSide
npm install
npm start
